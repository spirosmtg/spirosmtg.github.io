Welcome to GitHub Pages.

To create a git hub page i really simple.

1) Create a repository in your GitHub account with the name username.github.io or orgname.github.io.

2) Open the repository and go to settings in the right column.

3) Go to The Automatic Page Generator.

4) Click the Continue To Layouts button.

5) Preview your content in any theme you like.

6) When you find a theme that you like, click Publish.
